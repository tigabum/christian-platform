# QA Platform

A mobile application platform built with React Native that facilitates question-and-answer interactions between users and designated responders.

## Project Structure

The project consists of two main applications:

1. **User App (Main App)**

   - For users to ask questions and receive answers
   - Manage their questions and view responses
   - Search and filter functionality

2. **Responder App**
   - For designated responders to answer questions
   - Manage assigned questions
   - Track answer history

## Technology Stack

- React Native
- TypeScript
- Node.js/Express (Backend)
- MongoDB (Database)

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- React Native development environment
- MongoDB

- Answer history tracking
